About

This website is a simple project which has functionalities of a blog website.

This website has login/register functions that saves the information on a database. The database has hashing and salting implemented on the passwords of the users.

The design of the website uses HTML5 and CSS3 while the layout was made using Bootstrap. This is faster than doing the layout by scratch and I could focus more on the functionalities of the website.

Inside the website, we have forms for users to fill up for registration of an account and a contact page where I used SMTP (Simple Mail Transfer Protocol) in order for users to contact the developer. For the admin account, it has functionalities of adding, deleting and updating the blogs while the users can comment on the blogs posted.

I opted to add functionalities to my website rather than design as I am more comfortable programming the logic of an application.
